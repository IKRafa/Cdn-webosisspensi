# 📦 OSIS SPENSI - Content Delivery Network (CDN)

Selamat datang di repositori resmi aset Website OSIS SMP Negeri 7 Mengwi (SPENSI).

Repositori ini difungsikan secara khusus sebagai penyedia **Aset Statis dan Gambar (CDN)** untuk mendukung performa tinggi pada website utama OSIS SPENSI. Seluruh media, foto pengurus, ikon, dan berkas grafis lainnya didistribusikan dari sini melalui jaringan *jsDelivr*.

### 🚀 Mengapa Menggunakan Pendekatan Ini?
Kami memisahkan penyimpanan gambar dari kode sumber utama (HTML/CSS/JS) untuk memastikan:
- Waktu muat halaman (*page load time*) yang jauh lebih cepat bagi pengunjung.
- Penggunaan *bandwith* yang lebih efisien berkat teknologi cache *Edge* dari jsDelivr.
- Manajemen file media yang lebih terstruktur dan mudah dirawat.
